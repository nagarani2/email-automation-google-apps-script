# Email Automation System

This project automates email sending using Google Sheets, Gmail, and Google Apps Script.

## Features
- Send personalized emails to multiple recipients
- Replace {{Name}} placeholders automatically
- Mark emails as Sent after successful delivery
- Skip already sent emails
- Clear sent status
- Reset sheet data
- Custom Email Automation menu in Google Sheets

## Technologies Used
- Google Sheets
- Google Apps Script
- Gmail
